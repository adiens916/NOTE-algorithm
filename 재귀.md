- [10872. 팩토리얼](.\Baekjoon_Online_Judge\Step_by_step\09_Recursion\10872_팩토리얼.py)
  1. f(N)을 f(N - 1)로 구성해보기
     ```
     f(N)     = N * (N - 1) * (N - 2) * (N - 3) * ...
     f(N - 1) =     (N - 1) * (N - 2) * (N - 3) * ...
     => f(N)  = N * f(N - 1)
     ```
  2. 재귀식을 return 문에 적기
     ```
     if base_case:
         return base_value
     else:
         return N * f(N - 1)
     ```
